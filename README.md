# MusicPlayer
Music Player component for News Feed

## Features Implemented:
- [x] Light/Dark Modes
- [x] Portrait/Landscape Orientations
- [x] A11y: Padding/Content Description
- [x] Album and/or Artist(s)
- [x] Song track title/artist text overflow
- [x] Song Album Art default icon
- [x] Blur background color based on album art (double tap on album art)
- [x] Song track loop logic: OFF, ALL, ONE and corresponding player control enable/disable states
- [x] Like/Dislike button clicks
- [x] Track change animations
- [x] Empty State: No Tracks available
